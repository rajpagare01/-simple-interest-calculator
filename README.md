# Simple Interest Calculator

## Description
A simple interest calculator that takes the principal amount, rate of interest, and time as inputs to calculate the simple interest. This project is a final assignment demonstrating version control using Git and GitHub.

## Technologies Used
* Bash Scripting
* Git & GitHub
* Markdown

## Usage Instructions
1. Clone the repository.
2. Ensure the script has execute permissions: `chmod +x simple-interest.sh`
3. Run the script: `./simple-interest.sh`

## Example Calculation
```
Enter the principal:
1000
Enter rate of interest per year:
5
Enter time period in years:
2

The simple interest is: 100
```
