# Contributing to Simple Interest Calculator

Contributions are welcome! We appreciate your help in making this project better.

## Bug reports
If you find a bug, please open an issue and include as much detail as possible.

## Bug fixes
If you fix a bug, please submit a pull request with a clear description of the fix.

## Documentation improvements
Any improvements to the documentation are greatly appreciated!

## Enhancements
We welcome suggestions for enhancements and new features.

## Ideas and suggestions
If you have any ideas or suggestions, please open an issue to discuss them.
